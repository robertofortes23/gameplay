# GamePlay using React Native and Expo
### _Based on RocketSeat NLW Tougether Event_

Technologies used in the project:

- [React Native]
- [Expo]

## Installation

```sh
cd gameplay
npm i
expo start
```

   [Expo]: <https://expo.dev/>
